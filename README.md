# RICapture 📸📍

## 📌 Sobre o projeto

O RICapture é uma solução desenvolvida para coleta e análise de dados em campo na área industrial, utilizando o ecossistema Microsoft Power Platform.

## 🎯 Problema

O registro de evidências visuais em operações industriais era feito de forma manual, sem padronização e sem associação com localização geográfica, dificultando auditorias e análises.

## 💡 Solução

Desenvolvi o RICapture como uma solução integrada que permite capturar fotos com georreferenciamento via Power Apps, automatizar o fluxo de dados com Power Automate e analisar as informações em dashboards no Power BI.

## 🏗️ Arquitetura da solução

* Power Apps: interface para captura de fotos e dados em campo
* Power Automate: processamento automático e organização das informações
* Power BI: visualização e análise dos dados coletados

## ⚙️ Funcionalidades

* Captura de fotos diretamente pelo aplicativo
* Registro automático de localização (georreferenciamento)
* Armazenamento estruturado dos dados
* Automação do fluxo de informações
* Dashboard para análise e acompanhamento

## 📊 Resultados

* Padronização da coleta de dados em campo
* Redução de erros operacionais
* Melhoria na rastreabilidade das informações
* Apoio à tomada de decisão com dados visuais

## 🚀 Diferenciais

* Solução completa (coleta + automação + análise)
* Uso da Power Platform
* Aplicação real em ambiente industrial
* Escalável para diferentes cenários
